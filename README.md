# Real-Time_Crypto_Intelligence_Pipeline
Conception et implémentation d’une plateforme Data Engineering temps réel combinant streaming (Kafka + Spark Structured Streaming), batch (Airflow + FRED), architecture multi-sink (Data Lake + PostgreSQL/Supabase + Kafka + Redis) et intégration backend, avec gestion de schémas et préparation à la mise en production dans un environnement financier.
